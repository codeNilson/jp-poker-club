# components

Pasta para componentes reutilizáveis da interface.

Use esta área para peças pequenas e médias que podem aparecer em mais de uma tela, como botões compostos, cards, cabeçalhos, tabelas, badges e blocos de layout.

Subpastas esperadas neste projeto:
- `ui`: componentes base e de baixo nível, normalmente gerados ou adaptados do shadcn/ui.
- `layout`: estruturas de composição, como barras, sidebars, headers e shells de página.
- `forms`: componentes específicos de formulários, campos e combinações de entrada.
