# constants

Constantes compartilhadas.

Use esta pasta para valores fixos do domínio e da aplicação, como listas de tiers, labels, rotas internas, chaves de layout e configurações estáticas.

Se a constante estiver ligada a regra de negócio, esta pasta ajuda a manter o valor centralizado e fácil de revisar.

Esta pasta está vazia por enquanto. Constantes que aparecem em múltiplos arquivos devem ser movidas para cá.
