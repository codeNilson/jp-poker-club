# data

Dados estáticos ou mockados.

Use esta pasta para conteúdo que ainda não vem do banco ou da API, como exemplos, seeds de interface, listas de apoio e fixtures leves.

Quando o projeto crescer, este local também pode ajudar em protótipos e testes manuais de telas.

Esta pasta está vazia por enquanto. Todos os dados ativos são buscados diretamente do Supabase via `services/`.
